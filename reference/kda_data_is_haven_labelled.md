# Check whether all variables are haven-labelled and include variable/value labels

Check whether all variables are haven-labelled and include
variable/value labels

## Usage

``` r
kda_data_is_haven_labelled(data)
```

## Arguments

- data:

  A data frame.

## Value

A single logical value: `TRUE` if all variables are haven-labelled and
have both a variable label and value labels, otherwise `FALSE`.
