# YouAnalyser: Provide Standardized Analytics Products for YouGov DACH

A collection of functions that serve as tools for YouGov DACH Analytics
Partners. The function currently provides functions for Key Driver
Analysis. Planned addtions include Structural Equation Modeling,
Implicit Association Tests, Segmentation (Cluster Analysis), and
MaxDiff.
