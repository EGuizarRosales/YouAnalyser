# YouGov Colors

A named character vector containing the official YouGov colors with
their corresponding hex codes. Colors were sourced from the YouGov
Marketing Portal

## Usage

``` r
yougov_colors
```

## Format

### `yougov_colors`

A named character vector with 24 elements. Access each element using
e.g. `yougov_colors[["Red 1"]]` to get the hex code for "Red 1".
