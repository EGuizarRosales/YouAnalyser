# Exploratory data analysis: correlation

Compute and visualize correlations between variables.

## Usage

``` r
eda_correlation(
  data,
  variables = NULL,
  correlation_type = "pearson",
  correlation_args = list(),
  ggcorr_args = list(label = TRUE, label_size = 2, low = yougov_colors[["Blue 1"]], mid =
    "white", high = yougov_colors[["Red 1"]])
)
```

## Arguments

- data:

  A data frame.

- variables:

  Optional. A character vector of variable names to include in the
  correlation analysis. If `NULL`, all variables are used.

- correlation_type:

  Optional. Can be `"pearson"`, `"spearman"`, or `"tetrachoric"` (useful
  if variables are binary) among others. Check `method` argument in
  [`correlation::correlation()`](https://easystats.github.io/correlation/reference/correlation.html)
  to see all options. Defaults to `"pearson"`.

- correlation_args:

  Optional. A list of additional arguments passed to
  [`correlation::correlation()`](https://easystats.github.io/correlation/reference/correlation.html).

- ggcorr_args:

  Optional. A list of arguments passed to
  [`GGally::ggcorr()`](https://ggobi.github.io/ggally/reference/ggcorr.html).

## Value

A list with two elements: `d`, the correlation data frame, and `p`, a
ggplot object visualizing the correlation matrix.
