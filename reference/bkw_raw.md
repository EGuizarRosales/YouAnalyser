# BKW Employer Brand Positioning Study Data (Raw)

Raw data from the BKW Employer Brand Positioning Study conducted in 2026
(14713 BKW Zielgruppenumfrage Arbeitgeberpositionierung 2026). Key
Driver Analyses (linear regression with dominance analysis) were
performed. The data includes the dependent variable (F600) and 14
independent variables (F800_1 to F800_14) used in the regression model.

## Usage

``` r
bkw_raw
```

## Format

### `bkw_raw`

A data frame with 5,523 rows and 220 columns:

- RecordNo:

  Record number, starting from 0

- caseid:

  unique identifier for each case

- endtime:

  End time of the survey
